API Integration

Simple REST API

Send conversation data and receive comprehensive scoring analysis:

POST /analyze
Content-Type: application/json

{
  "conversation": [
    {
      "role": "user",
      "content": "Act as a marketing expert..."
    },
    {
      "role": "assistant", 
      "content": "Here's your marketing plan..."
    }
  ],
  "userId": "user-123",
  "metadata": {
    "platform": "your-ai-platform"
  }
}

Response:

{
  "overallScore": 78,
  "proficiencyLevel": "Advanced",
  "dimensionScores": {
    "promptEngineering": 20,

    "problemSolving": 19,
